## Code Style

- Formatter: Ruff
- Docstrings: Google style
- Type hints: Required on all public functions. Document complex types (array shapes, generic parameters) in docstrings.
- Imports: Standard library, then third-party, then local (blank line between groups)

## Naming

- Files: `snake_case.py`
- Classes: `PascalCase`
- Functions/variables: `snake_case`
- Constants: `UPPER_SNAKE_CASE`
